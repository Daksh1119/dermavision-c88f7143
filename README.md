# DermaVision

DermaVision is an AI-powered skin disease detection web application built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui. It integrates Supabase for authentication, data storage, and edge functions.

## Tech Stack
- Vite + React + TypeScript
- Tailwind CSS + shadcn-ui
- Supabase (Auth, DB, Storage)
- Radix UI
- React Query

## Development
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Environment
Create a `.env` file based on `.env.example` and set all required `VITE_` variables.
